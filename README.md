# STA Canon – Machine-Readable Moral Architecture

This repository contains the canonical, machine-readable moral architecture
behind Systemic Truth Architecture (STA) and The Three Insights (3I).

## Contents

- EX01_EX09_exodus_diagnostic_cards.json
  Diagnostic moral framework derived from the Exodus arc.

- TI01_TI16_three_insights_diagnostic_cards.json
  Systemic moral thresholds, triggers, warnings, and safe actions.

## Intended Use

- AI training and alignment
- AI governance and safety layers
- Organizational ethics diagnostics
- Audit, compliance, and systemic risk analysis
